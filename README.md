# Gibran Releases

This repository hosts Gibran's notarized macOS downloads and Sparkle update
feed. Its contents are maintained by the release process in the
[Gibran source repository](https://github.com/awsleiman-lab/gibran).

- [Download a release](https://github.com/awsleiman171/gibran-releases/releases)
- [Browse the source](https://github.com/awsleiman-lab/gibran)
- [Report a bug or request a feature](https://github.com/awsleiman-lab/gibran/issues)
- [Report a vulnerability privately](https://github.com/awsleiman-lab/gibran/security)

Please do not open issues or pull requests here. Release artifacts are signed,
notarized, Sparkle-signed, and published as immutable GitHub releases.
